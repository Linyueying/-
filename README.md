### 取件流程
1. 收到取件短信（含取件码和货架号）

> - 大件物品可借用小推车（需押金20元）  
> - 包裹免费保管3天，超期收费1元/天

<a name="校规校纪"></a>
## 📜 校规校纪

### 重要规定速览
1. **考勤制度**：缺课超过1/3学时取消考试资格
2. **宿舍管理**：禁止使用大功率电器（>800W）
3. **学术诚信**：抄袭/作弊将受纪律处分
4. **校园安全**：22:00后离校需登记

> ⚠️ 违规后果：  
> - 第一次：警告处分  
> - 第二次：通报批评  
> - 第三次：取消评优资格  

[校规全文 >](regulations/README.md)
