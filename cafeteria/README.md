# 🍽️ 食堂指南

| 食堂名称 | 位置 | 开放时间 | 特色 |

## 推荐菜品

### 第一食堂
- **早餐**：葱油拌面、小笼包、豆浆油条
- **午餐**：红烧排骨套餐、麻辣香锅
- **晚餐**：牛肉面、水饺

### 风味餐厅
- 重庆小面（辣度可选）
- 广式烧腊（烧鸭饭、叉烧饭）
- 西北拉面（现拉现煮）
